# Free Tier Version - תמלול חינמי

## הגבלות גרסת Free
- **גודל קובץ מקסימלי:** 15MB (כ-15 דקות הקלטה)
- **מספר קבצים:** 1 קובץ בכל פעם
- **תמלולים מקביליים:** 1 תמלול בכל פעם בשרת
- **זמן timeout:** 10 דקות מקסימום

## שינויים מהגרסה המלאה
1. הפחתת גודל קובץ מ-500MB ל-15MB
2. הגבלת מספר קבצים מ-∞ ל-1
3. הפחתת timeout מ-60 דקות ל-10 דקות
4. הפחתת מקביליות מ-3 ל-1

## הוראות Deploy
1. עבור ל-branch `free-tier`
2. עשה Deploy ב-Render Free Tier
3. הוודא ש-FFmpeg זמין
4. בדוק שהשירות עובד עם קבצים קטנים

## מתאים ל:
- Render Free Tier (0.1-0.5 CPU, 512MB RAM)
- 15 דקות timeout
- בחינם לחלוטין
